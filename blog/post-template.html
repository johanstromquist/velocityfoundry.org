How to Use This Template

To create a new blog post:

  1. Copy this file and rename it to match your slug (e.g., 2025-01-my-post-slug.html)
  2. Update the meta description and title in the head
  3. Add an entry to blog/posts.js with:
    • slug (must match your filename without .html)
    • title
    • date (for display)
    • publishDate (optional - for scheduling)
    • excerpt
    • readTime
    • image (path to hero image)
  4. Replace this content with your post
  5. Commit and push to GitHub

What's Dynamic?

The following elements are automatically generated from posts.js:

  • Navigation bar - Edit in blog-layout.js
  • Hero section - Title, date, and background image from posts.js
  • Next/Previous links - Automatically generated based on post order
  • Footer - Edit in blog-layout.js
